# AsyncImg

Load images asynchronously in HTML.

## Usage
Use `async-src` instead of `src` on your images.

```html
<img async-src="/path/to/image.jpg" alt="...">
```

Load the library:
```html
<script src="https://cdn.jsdelivr.net/gh/rickwierenga/AsyncImg@master/asyncimg.js"></script>
```
